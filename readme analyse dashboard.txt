یک صفحه داشبورد تحلیلی و گزارش‌دهی با استفاده از PHP، HTML، CSS و JavaScript بساز. این صفحه باید دارای ویژگی‌ها و ساختار زیر باشد:۱. ساختار کلی و استایل:کل صفحه در یک فایل PHP واحد باشد.از کتابخانه Bootstrap 5 برای طراحی واکنش‌گرا (responsive) و استایل‌دهی کلی استفاده شود.تمام بخش‌های اصلی محتوا (مانند پنل فیلتر، نمودارها و جداول) در کانتینرهای کارتی (<div class="card content-card">) قرار گیرند.کارت‌ها باید دارای card-header برای عنوان و card-body برای محتوا باشند.۲. پنل فیلتر:در بالای صفحه، یک کارت برای فیلتر کردن گزارش‌ها قرار داشته باشد.این پنل باید شامل موارد زیر باشد:دو فیلد ورودی تاریخ شمسی (persian-datepicker) برای انتخاب بازه زمانی «از تاریخ» و «تا تاریخ».چند فیلد select (دراپ‌داون) برای فیلتر کردن داده‌ها (مثلاً بر اساس نوع دستگاه، نام دستگاه و...).یک دکمه اصلی با عنوان «ایجاد گزارش» برای ارسال فرم.یک دکمه ثانویه با عنوان «چاپ» برای پرینت کردن گزارش.۳. نحوه عملکرد و دریافت داده:با کلیک روی دکمه «ایجاد گزارش»، صفحه نباید به صورت کامل رفرش شود.داده‌ها باید به صورت ایجکس (AJAX) با استفاده از jQuery $.getJSON از یک فایل API جداگانه (مثلاً api/api_get_analytics_data.php) دریافت شوند.در حین بارگذاری داده‌ها، یک اسپینر لودینگ به کاربر نمایش داده شود.پس از دریافت موفق داده‌ها، محتوای گزارش (نمودارها و جداول) در زیر پنل فیلتر نمایش داده شود.۴. نمایش محتوای گزارش:برای نمایش داده‌های بصری از کتابخانه Chart.js استفاده شود.انواع مختلفی از نمودارها باید پشتیبانی شوند:نمودار دایره‌ای (Pie Chart)نمودار میله‌ای (Bar Chart)نمودار خطی (Line Chart)نمودار پارتو (ترکیبی از میله‌ای و خطی)داده‌های جدولی باید در یک <table> با استایل Bootstrap نمایش داده شوند.۵. قابلیت چاپ:صفحه باید دارای استایل‌های مخصوص چاپ (@media print) باشد.در حالت چاپ، فقط محتوای گزارش (کارت‌های نمودار و جدول) نمایش داده شود و پنل فیلتر، منوها و دکمه‌ها مخفی شوند.کل محتوای گزارش باید در یک قالب شبیه به صفحه A4 قرار گیرد.نمودارها باید به صورت خودکار به تصویر تبدیل شده و در نسخه چاپی نمایش داده شوند.مثال درخواست:"یک داشبورد گزارش‌دهی برای تحلیل آمار تولید بساز که دارای فیلتر تاریخ و دستگاه باشد و نمودارهای تولید روزانه و تولید بر اساس هر دستگاه را نمایش دهد. لطفاً از استایل و ساختار مشخص شده در پرامپت داشبورد تحلیلی استفاده کن."
